질환명 (한글) 무월경증(원발성) 질환명 (영문) Primary Amenorrhea 
분류 : 비뇨생식기계 질환 
상병코드 : N910 
정의 : 사춘기가 지난 16세 이상의 젊은 여성에게서 월경(초경)이 나타나지 않는 증상을 말한다. 
원인 : 알려져 있지 않으나 다음의 원인에 의할 것이라 여겨진다. 1) 선천적 기형 : 여성생식기(질, 자궁, 난소)가 없거나 비정상적으로 형성됨 처녀막이 폐쇄되어 월경혈이 나오지 못함. 2) 내분비계의 질환(종양, 감염, 성장결핍) 3) 염색체 질환, 정서적 불안 4) 식사습관에 문제가 있는 경우 : 비만, 식욕항진, 신경성 식욕부진, 과다한 다이어트 5) 정신에 작용하는 약물이나 호르몬제를 복용한 경우 6) 운동량이 큰 체육활동에 참가하는 경우 7) 초경이 시작되기 전 성교를 통해 임신한 경우 
증상 : 대부분의 소녀들이 14세 전후에서 월경을 시작하는데 비해 사춘기가 지나도 월경이 시작되지 않는 경우를 말한다. 
치료 : 약물요법 월경을 유도하기 위해 호르몬(프로게스테론)제를 투여한다. 호르몬 투여 중지후 출혈이 시작되면, 생식기계가 정상기능을 하고, 생식기관도 열려 있으며 뇌하수체 질환이 없음을 의미한다. 호르몬 투여 중지후에도 출혈이 일어나지 않으면 성선자극호르몬을 다시 투여해야 한다. 
기타 : 일반요법 정서적으로 스트레스 문제가 있을 경우에는 가족이나 친구, 전문상담가와 상담하여 해결하도록 함 정신변화를 일으키는 약물을 삼가한다.
